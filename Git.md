GitHub por terminal
=========

Clonar repositorio
```
git clone git@github.com:rulz/myproject.git
```

status
```
git status
```

add all
```
git add .
```

commit
```
git commit -am "poner el comentario de lo que hiciste"
```

push (enviar)
```
git push
```



pull (actulizar o sea baja cambios)
```
git pull
```
